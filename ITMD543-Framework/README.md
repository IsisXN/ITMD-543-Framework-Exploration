# My Portfolio – React + Vite Project

Project Description
This project is a personal portfolio website built using **React** and **Vite**.  
It includes three main pages:

- **Home** – Introduces the project, provides a featured photo, and highlights lessons learned during development.
- **Photography** – Displays a 2x3 photo gallery with hover effects, showcasing pictures from Spain, Italy, and China.
- **About Me** – Shares background information and campus involvement.

The goal of this project was to learn how to structure a react application, manage components, use react router for navigation, and apply consistent custom CSS styling across multiple pages.  
The site features a navy blue color theme with white and light blue accents and includes a custom navigation bar.


 Technologies & Frameworks Used
- **React** (frontend library)
- **Vite** (build tool + development environment)
- **React Router v6** (page navigation)
- **Custom CSS** (styling and layout)
- **Node.js + npm** (package management)
